Description

Starting the game, players will see an animated welcome screen as below and can choose to skip

Mine Seeker's welcome screen
Then, you will see the main screen where you can learn how to play, adjust the game level or start playing:
Mine Seeker's main screen
The rule is simple: your goal is to find all the hidden mines by revealing the rectangles. Everytime you reveal a rectangle, you either find a hidden mine or you get a hint of how many hidden mines are left on the row/column of that rectangle.
Mine Seeker's gameplay
To adjust the difficulty-level, you can go to the options screen to adjust the size of the grid and the number of hidden mines
Mine Seeker's options screen
That's all about the game, you can clone this project by pressing the GitHub icon above and run it on an Android device with API level at least 29 to play this game. Have fun!
