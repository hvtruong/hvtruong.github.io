Projects


Toxic comment classification challenge

The purpose of this project is to help reduce toxicity in the online environment by detecting abusive comments. To do this, we created a multi-lable classification model using LSTM to classify the comments by probability with 6 labels: toxic, severe toxic, obscene, threat, insult, identity hate.

Gitlab Analyzer

This is a web application project created to analyze the code contributions of team members in GitLab repositories, and report scores, metrics about them. This is designed to analyze unlimitted repositories simultaneously, allowing users to share the analysis with others, and to adjust the scores on their own.

Safe Restaurants

This is an Android application project created to for finding safe restaurants based on hazard levels. The application uses latest routine inspections at the restaurants to determine their levels. The restaurants are displayed as colored markers on Gooogle Maps representing their hazard levels so users can choose nearby ones. Moreover, users can also click the markers to see detailed violations.

Digit Recognizer

This is a personal machine learning project aims to recognize digits from 0 to 9 using convolution neural network. The project used dataset from the challenge Kaggle Digit Recognizer and obtained an accuracy of 99.564%. Digit Recognizer is a promising project as it can be really useful in areas like reading cheques, license plates, or handwritten numbers.

Mine Seeker

This is a personal assignment creating an Android game similar to the famouse Microsoft Minesweeper. However, the goal of this game is to find all the hidden mines rather avoiding them. Users can adjust the size of the grid and the numbers of mines to modify difficulty-level.
